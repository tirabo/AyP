Introducción a Matplotlib#

Referencias

  • https://matplotlib.org/stable/tutorials/introductory/pyplot.html

  • https://www.tutorialspoint.com/numpy/numpy_matplotlib.htm

  • http://www.denebola.org/japp/CC/graficos.html

Matplotlib es una biblioteca de dibujo para Python. Se utiliza junto con NumPy para proporcionar un entorno que es una alternativa eficaz de código abierto para MatLab. También se puede utilizar con kits de herramientas gráficas como PyQt y wxPython.

El módulo Matplotlib fue escrito por primera vez por John D. Hunter. Desde 2012, Michael Droettboom es el desarrollador principal. El paquete está disponible en distribución binaria así como en forma de código fuente en www.matplotlib.org.

Convencionalmente, el paquete se importa en el script de Python añadiendo la siguiente sentencia

Si pyplot no está instalado se debe hacer:

pip install matplotlib

Un vez instalado importamos pyplot con la siguiente instrucción:

from matplotlib import pyplot as plt

pyplot es el módulo más importante de la biblioteca matplotlib, que se utiliza para dibujar datos 2D.

Siempre debemos importar numpy que es la biblioteca que usaremos para manejar los datos a ser representados.

import numpy as np

El siguiente script dibuja el gráfico de la función \(y = x^2 + 5\):

from matplotlib import pyplot as plt
import numpy as np

x = np.arange(1,11,0.5)
y =  x**2 + 5
print(x)
print(y)
plt.title("Demostración de Matplotlib")
plt.xlabel("leyenda del eje x")
plt.ylabel("leyenda del eje y")
plt.plot(x,y)
plt.show()

Se crea un objeto ndarray x a partir de la función np.arange() que serán los valores del eje x. Los valores correspondientes en el eje y se almacenan en otro objeto ndarray y. Estos valores se dibujan utilizando la función plot() del submódulo pyplot del paquete matplotlib.

La representación gráfica se muestra mediante la función show().

Observar que para dibujar o plotear se introduce una “pareja de datos”, primero el eje \(x\) y luego los correspondientes valores en el eje \(y\). La cantidad de datos en \(x\) debe ser igual a la cantidad de datos en \(y\).

Hay diferentes opciones de ploteo, que no exploraremos mayotrmente aquí. Por ejemplo,

x = np.arange(1,21, 0.5)
y =  2 * x**2 + 3
plt.title("Demostración de Matplotlib")
plt.xlabel("leyenda del eje x")
plt.ylabel("leyenda del eje y")
plt.plot(x,y, "or")
plt.show()

En el código anterior “or” nos dice que se deben dibujar “dots” de color rojo.

1. Dibujar varias parejas de datos#

Se pueden graficar varias funciones en el mismo ploteo introduciendo varias parejas de datos.

Veamos un ejemplo:

def f1(x):
    # x es array numpy de números
    y = np.sin(x)
    return y

def f2(x):
    # x es array numpy de números
    y = np.sin(x) + np.sin(5.0*x)
    return y

def f3(x):
    # x es array numpy de números
    y = np.sin(x) * np.exp(-x/10.)
    return y

x = np.arange(0, 10*np.pi, 0.1)


plt.plot(x, f1(x), x, f2(x), x, f3(x))
plt.show()

Se pueden poner opciones de ploteo a cada pareja, por ejemplo (aunque no queda muy bien que digamos)

x = np.arange(0, 10*np.pi, 0.25)
plt.plot(x, f1(x), "or", x, f2(x), x, f3(x), ".b")
plt.show()

2. La función subplot()#

La función subplot() permite trazar diferentes gráficos en la misma figura, pero como dibujos separados.

En el siguiente script, se grafican el seno y el coseno.

# Calcular las coordenadas x e y de los puntos de las curvas seno y coseno
x = np.arange(0, 3 * np.pi, 0.1)
y_sin = np.sin(x)
y_cos = np.cos(x)

# Configurar una grilla tenga ancho 2 y altura 1 (se graficará un plot al costado de otro),
# y establecer la primera de esas subtramas como activa.
plt.subplot(1, 2, 1)

# Se hace el primer plot
plt.plot(x, y_sin)
plt.title('Seno')

# Establecer el segundo subplot como activo y hacer el segundo plot.
plt.subplot(1, 2, 2)
plt.plot(x, y_cos)
plt.title('Coseno')
plt.show()

La función subplot() de pyplot es utilizada para crear grillas de subplots en una figura de Matplotlib. Esta función acepta tres parámetros posicionales: nrows, ncols e index, y opcionalmente otros parámetros para ajustar el tamaño, el espacio y los límites de los subplots. A continuación se describen los tres primeros parámetros de la función subplot():

  • nrows: número de filas de la grilla de subplots.

  • ncols: número de columnas de la grilla de subplots.

  • index: índice del subplot actual, comenzando desde 1 y aumentando de izquierda a derecha y de arriba hacia abajo.

Existesn otros argumentos y parámetros para compartir ejes y para ajustar el tamaño, el espacio y los límites de los subplots.

Es decir el método subplot() permite hacer una grilla \(m \times n\) y dibujar en cada rectángulo de esa grilla.

def f1(x):
    y = np.sin(x)
    return y

def f2(x):
    y = np.sin(x) + np.sin(5.0*x)
    return y

def f3(x):
    y = np.sin(x) * np.exp(-x/10.)
    return y

def f4(x):
    y = np.cos(x) * x
    return y

plt.subplot(2, 2, 1)
plt.plot(x, f1(x))
plt.title('sen(x)')

plt.subplot(2, 2, 2)
plt.plot(x, f2(x))
plt.title('sen(x) + sen(5x)')

plt.subplot(2, 2, 3)
plt.plot(x, f3(x))
plt.title('sen(x)*exp(-x/10)')

plt.subplot(2, 2, 4)
plt.plot(x, f4(x))
plt.title('cos(x)*x')

plt.show()

No quedó muy bien, pero se puede arrglar con un truco:

plt.subplot(3, 2, 1)
plt.plot(x, f1(x))
plt.title('sen(x)')

plt.subplot(3, 2, 2)
plt.plot(x, f2(x))
plt.title('sen(x) + sen(5x)')

plt.subplot(3, 2, 5)
plt.plot(x, f3(x))
plt.title('sen(x)*exp(-x/10)')

plt.subplot(3, 2, 6)
plt.plot(x, f4(x))
plt.title('cos(x)*x')

plt.show()

Aunque debe haber mejores formas de hacerlo.

3. Trabajando con texto dentro del gráfico#

Exiten funciones para añadir texto (etiquetas) a los ejes de la gráfica y a la gráfica en sí. Veamos algunos ejemplos:

x = np.arange(0, 5, 0.05)
p = plt.plot(x,np.log10(x)*np.sin(x**2)) # log(x)*sen(x**2)
plt.xlabel('Eje X')        # Etiqueta del eje X
plt.ylabel('Eje Y')        # Etiqueta del eje Y
plt.title('Mi grafica')    # Título del gráfico
plt.text(1, -0.4, 'Nota')  # Texto en coodenadas (1, -0.4)
plt.show()

En este ejemplo, se usó la función text() para añadir un texto arbitrario en la gráfica, cuya posición se debe dar en las unidades de la gráfica. Cuando se utilizan textos también es posible usar fórmulas con formato LaTeX. Veamos un ejemplo,:

plt.clf()   #limpio la figura
x = np.arange(0.1, 6*np.pi, 0.1)
y1 = np.sin(x)/x
y2 = np.sin(x)*np.exp(-x)
plt.plot(x, y1, x, y2)
plt.text(2, 0.6, r'$\frac{\operatorname{sen}(x)}{x}$', fontsize=22)
plt.text(13, 0.2, r'$\operatorname{sen}(x) \cdot e^{-x}$', fontsize=16)
plt.grid()       # Añado una malla al gráfico
plt.title('Representacion de dos funciones')
plt.xlabel('Tiempo / s')
plt.ylabel('Amplitud / cm')
plt.show()

Aquí hemos usado código LaTeX para escribir fórmulas matemáticas, para lo que siempre hay que escribir entre r'$formula$' y hemos usado un tamaño de letra mayor con el parámetro fontsize. En la última línea hemos añadido una grilla con la función grid().

Nota. LaTeX es un sistema de escritura orientado a contenidos matemáticos muy popular en matemática, física teórica y ciencias de la computación.

4. Histogramas#

Cuando tenemos un conjunto de datos numéricos, por ejemplo como consecuencia de la medida de una cierta magnitud y queremos representarlos gráficamente para ver la distribución subyacente de los mismos se suelen usar los gráficos llamados histogramas.

Los histogramas representan el número de veces que los valores del conjunto caen dentro de un intervalo dado, frente a los diferentes intervalos en los que queramos dividir el conjunto de valores.

En Python podemos hacer histogramas muy fácilmente con la función hist() indicando como parámetro un array con los números del conjunto. Si no se indica nada más, se generará un histograma con 10 intervalos (llamados bins, en inglés) en los que se divide la diferencia entre el máximo y el mínimo valor del conjunto. Veamos un ejemplo:

nums = np.random.randn(20000)  # array con 200 números aleatorios en distribución normal
# print(nums)

plt.hist(nums, color='green', rwidth=0.90)
plt.show()

Si no se le proporciona ningún otro argumento a randn() produce floats alrededor de 0 y con una varianza = 1.

Vemos que los números del array se dividieron automáticamente en 10 intervalos (o bins) y cada barra representa para cada una de ellos el número de valores que caen dentro. Si en lugar de usar sólo 10 divisiones queremos usar 21 por ejemplo, debemos indicarlo como un segundo parámetro:

plt.hist(nums, bins=30 , rwidth=0.85)
plt.show()

El la figura de abajo se muestra el resultado de superponer ambos histogramas y luego hacemos otro gráfico.

plt.hist(nums, color='green', rwidth=0.85)
plt.hist(nums, bins=20 , color= 'blue', rwidth=0.85)
plt.show() # dibuja el gráfico y podemos empezar otro
x = np.arange(0.1, 6*np.pi, 0.1)
y1 = np.sin(x)/x
y2 = np.sin(x)*np.exp(-x)
plt.plot(x, y1, x, y2)
plt.show()

La función hist() devuelve una tupla con tres elementos, que son un array con el número elementos en cada intervalo, un array con el punto del eje X donde empieza cada intervalo y una lista con referencias a cada una de las barras para modificar sus propiedades (consulten el manual de matplotlib para encontrar más información y mayores posibilidades de uso).

x = plt.hist(nums, bins=20 , color= 'blue', rwidth=0.85)
print('Cantidad de valores del 10º intevalo:',x[0][9]) # cuantos valores están en el 10º intervalo

Cantidad de valores del 10º intevalo: 2797.0

5. Gráficos de barras#

El submódulo pyplot proporciona la función bar() para generar gráficos de barras. El siguiente ejemplo produce el gráfico de barras de dos àrreglos, x e y.

x = np.array([5,8,10])
y = np.array([12,16,6])

x2 = [6,9,11]
y2 = [6,15,7]
plt.bar(x, y, align = 'center')
plt.bar(x2, y2, color = 'g', align = 'center')
plt.title('Gráfico de barras')
plt.ylabel('Eje y')
plt.xlabel('Eje x')
plt.show()

7. Lectura y escritura de archivos con numpy#

Los objetos ndarray (los array de NumPy) se pueden guardar y cargar. Las funciones de entrada y salida disponibles son

  • las funciones load() y save() manejan archivos binarios numPy (con extensión npy),

  • las funciones loadtxt() y savetxt() manejan archivos de texto.

NumPy introduce un formato de archivo simple para los objetos ndarray. El archivo .npy almacena los datos, la forma, el dtype y otra información requerida para reconstruir el ndarray, de manera que el array sea recuperado correctamente aunque el archivo esté en otra máquina con diferente arquitectura.

numpy.save()

El método save() almacena el array de entrada en un archivo con extensión npy.

arr_1 = np.array([1,2,3,4,5])
np.save('archivo_salida',arr_1)

Para reconstruir el array desde archivo_salida.npy, utiliza la función load().

arr_b = np.load('archivo_salida.npy')
print(arr_b)

[1 2 3 4 5]

Producirá la siguiente salida

array([1, 2, 3, 4, 5])

Las funciones save() y load() aceptan un parámetro booleano adicional allow_pickles. Un pickle en Python se utiliza para serializar y des-serializar objetos antes de guardar o leer de un archivo de disco.

savetxt()

El almacenamiento y recuperación de datos de arreglos en formato de texto se realiza con las funciones savetxt() y loadtxt().

Ejemplo:

arr_a = np.array([1,2,3,4,5])
np.savetxt('out.txt',arr_a)
arr_b = np.loadtxt('out.txt')
print(arr_b)

[1. 2. 3. 4. 5.]

Producirá la siguiente salida

[1., 2., 3., 4., 5.]

Las funciones savetxt() y loadtxt() aceptan parámetros opcionales adicionales como la cabecera, el pie de página y el delimitador.

8. Representando datos experimentales#

Cuando estamos en el laboratorio (u observatorio) haciendo un experimento y midiendo algún parámetro físico solemos obtener un conjunto de números que normalmente guardaremos en un fichero de texto en cualquier ordenador. Representar datos leídos de un fichero en lugar de generarlos directamente como hemos hecho hasta ahora, es tan sencillo como leer los datos y pasarlos a arrays de numpy. Una vez hecho, se representan gráficamente como ya hemos visto. Supongamos que tenemos los resultados de un experimento en una tabla de dos columnas almacenada en el fichero “datos_2col.txt”.

Primero creemos el archivo (no tenemos uno disponible)

col1 = np.random.randint(-20, 20, 100)
col2 = np.random.randint(0, 40, 100)
with open('datos_2col.txt', 'w') as f:
    for i in range(100):
        f.write(str(col1[i])+'\t'+str(col2[i]) + '\n')

Para poder dibujarlos en un gráfico hacemos,:

# Leemos el fichero de dos columnas de datos, pasándolo a un array
datos = np.loadtxt('datos_2col.txt')
print(datos.shape)
